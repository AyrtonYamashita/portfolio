Automação de Processos

Desenvolvimento de uma aplicação de automação de processos administrativos, analisei um processo manual com muitas repetições e construí a solução para este problema utilizando Python como um gerenciador dos dados recebidos.
A aplicação lê uma quantidade massiva de dados em um arquivo .xlsx e replica esses dados no sistema iniciando um navegador automáticamente, ao fim da operação é gerado um relatório de todos documentos enviados e não enviados.
